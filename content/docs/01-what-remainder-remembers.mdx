Modulo is an operation that returns the reminder of a division

```
13 / 5 = 2 (and remainer of 3)
```

You can see that we are doing some weird things, you can probably think why not just calculate it like

```
13 / 5 = 2.6
```

And get clean value with decimal fraction?
